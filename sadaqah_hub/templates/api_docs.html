API documentation

  • GET /api/charities — list charities
  • GET /api/charities/<id> — charity detail with reviews
  • GET /api/charities/<id>/reports — report URLs
  • POST /api/charities — create charity (requires passphrase header)
  • POST /api/charities/<id>/reviews — create review (requires passphrase header)

For write endpoints send X-Community-Passphrase.
